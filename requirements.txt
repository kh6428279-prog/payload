# متطلبات تشغيل البرنامج
# هذا البرنامج يستخدم مكتبات Python المدمجة فقط
# لا توجد مكتبات خارجية مطلوبة

# متطلبات النظام:
# - Python 3.6 أو أحدث
# - Android SDK (لإنشاء ملفات APK)
# - Gradle (اختياري - البرنامج يحاول استخدام gradlew المدمج)

# المكتبات المستخدمة (مدمجة في Python):
# - tkinter: لواجهة المستخدم الرسومية
# - threading: للتعامل مع الخيوط المتعددة
# - socket: للاتصال بالشبكة
# - os: للتعامل مع نظام الملفات
# - subprocess: لتشغيل العمليات الفرعية
# - json: للتعامل مع ملفات JSON
# - typing: لنوع البيانات (Python 3.5+)

# ملاحظة: تأكد من أن Python 3.6+ مثبت على النظام
